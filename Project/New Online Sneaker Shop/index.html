The Online Sneaker Shop activity network model consists of the object types "SneakerRequestHandler", "SizeRequestHandler", and "CheckoutRequestHandler" the event type/node "SneakerRequest", and the activity types/nodes "HandleSneakerRequest", "HandleSizeRequest" and "HandleCheckoutRequest". It's dynamics is described by the DPMN process diagram on the right.

Default scenario

The default scenario has 4 request handlers (with IDs 1–4), 6 size handlers (with IDs 11–16), 10 size channels, and 15 checkout channels.";

For all three scenario the number of sizes available were 60 for size 1, 20 for size 2 and 30 for size 3. Stock numbers can be changed in sim.model.setupStatistics .
